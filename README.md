# Democracy and Economic Growth: Stata Empirical Project

This repository contains a Stata-based empirical project examining the relationship between democracy and real GDP per capita growth using a country-year panel dataset.

The main purpose of this repository is to present a reproducible empirical workflow, including data cleaning, panel construction, regression analysis, robustness checks, and generated tables and figures.

## Repository Structure

```text
code/
    final_code.do

data/
    raw/
        REAL GDP.csv
        Tradeopenness.csv
        GFCF.csv

    clean/

        figure_2_lpoly_growth_democracy.png

    logs/

paper/
    Econometrics Empirical paper.pdf
```

## How to Run

Open Stata and set the working directory to the repository root:

```stata
cd "/your/path/to/Econometric-empirical-paper"
do code/final_code.do
```

The do file assumes that the working directory is the repository root. It uses relative paths to read raw data and export generated outputs.

## Main Files

- `code/final_code.do`: Main Stata do file for data cleaning, panel construction, regressions, robustness checks, and output generation.
- `data/raw/`: Raw input datasets.
- `data/clean/final_country_year_panel.dta`: Final cleaned country-year panel generated from the raw data.
- `output/tables/table_outcome.pdf`: Combined table output from the empirical analysis.
- `output/figures/`: Generated figures.
- `output/logs/final code.log`: Log file from a successful run of the Stata workflow.
- `paper/democracy_growth_writing_sample.pdf`: Polished writing sample based on the empirical project.

## Writing Sample

The writing sample in `paper/` provides the broader research context for this empirical project. It includes the research motivation, data description, model discussion, interpretation of results, and limitations of the analysis.

The main purpose of this repository, however, is to demonstrate the reproducible Stata workflow behind the project. The writing sample is included as supplementary material for readers who want to see the full research paper.

## Notes

The raw data are from publicly available sources, including the World Bank and Our World in Data. The cleaned dataset, generated tables, figures, and log file are included for quick inspection, but the main outputs can be regenerated by running the Stata do file.
